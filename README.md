## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Dayaftyesterday/portfolio.git
   ```

2. Open `index.html` in your browser

## Adding Your Images

Replace the placeholder images in the `<img>` tags with your actual photo URLs:

1. Create an `images` folder in your repository
2. Upload your photos to GitHub
3. Update the `src` attributes in index.html with your image paths:
   ```html
   <img src="images/your-photo.jpg" alt="Description">
   ```

## Customization

- **Colors**: Edit CSS variables in `:root` selector
- **Fonts**: Change Google Fonts links in `<head>`
- **Content**: Update text in HTML sections
- **Layout**: Modify grid classes in gallery section
